# E9-222: Signal Processing in Practice

## Course Contents

- **Lecture 1:** Linear and circular convolution

## Quick Start

### Prerequisites

- Python 3.8+
- pip or conda

Recommended: Have a local installation

### Clone the Repository

```bash
git clone https://github.com/spectrum-lab-iisc/e9-222-signal-processing-in-practice.git
cd e9-222-signal-processing-in-practice
```

### Set Up Environment

**Option 1: Using venv**
```bash
python -m venv .venv
source .venv/bin/activate 
# On Windows: .venv\Scripts\activate
```

**Option 2: Using conda**
```bash
conda create -n signal-processing python=3.10
conda activate signal-processing
```

### Install Dependencies

```bash
pip install numpy matplotlib scipy
```
